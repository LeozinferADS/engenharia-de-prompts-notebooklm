# Engenharia de Prompts com NotebookLM

## 📚 Sobre o Projeto

Este repositório foi desenvolvido como parte do desafio da DIO sobre Engenharia de Prompts utilizando o NotebookLM.

O objetivo é demonstrar o uso da Inteligência Artificial como ferramenta de apoio aos estudos, reunindo materiais, resumos e exemplos de prompts para facilitar o aprendizado.

---

## 🎯 Objetivos

- Compreender os conceitos de Engenharia de Prompts.
- Aprender boas práticas na criação de prompts.
- Utilizar o NotebookLM para organizar conhecimentos.
- Criar um material de consulta para estudos futuros.

---

## 📖 Fontes Utilizadas

- Curso Introdução à Engenharia de Prompts (DIO)
- NotebookLM (Google)
- Documentação da OpenAI

---

## 💬 Exemplos de Prompts

**Pergunta**

**Resultado**

---

## 📌 Glossário

- **Prompt:** instrução enviada para a IA.
- **Contexto:** informações que ajudam a IA a compreender a tarefa.
- **Persona:** papel que a IA deve assumir.
- **Iteração:** melhoria contínua dos prompts.

---

## ✅ Conclusão

Durante este desafio foi possível compreender como a Engenharia de Prompts influencia diretamente a qualidade das respostas geradas por Inteligências Artificiais, tornando o processo de estudo mais eficiente.
